## Clone the repository

```
git clone https://github.com/yuzakki/Job-listings-with-filtering.git
```

Navigate to the project directory:

```bash
cd Job-listings-with-filtering
```

Install dependencies:

```bash
npm install
```

## Usage

Start the development server:

```bash
npm run dev
```

then go to
`http://localhost:5173/`
